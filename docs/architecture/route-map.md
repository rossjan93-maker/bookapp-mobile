# Route Map

## Fixed route structure
- `app/(auth)/login.tsx`
- `app/(tabs)/index.tsx`
- `app/(tabs)/search.tsx`
- `app/(tabs)/library.tsx`
- `app/(tabs)/notes.tsx`
- `app/(tabs)/profile.tsx`
- `app/book/[id].tsx`
- `app/user-book/[id].tsx`
- `app/settings/import.tsx`

## Route purpose
- `/(auth)/login`: auth/onboarding shell entry.
- `/(tabs)`: primary app shell for core daily use areas.
- `/book/[id]`: book detail placeholder.
- `/user-book/[id]`: user-book relation detail placeholder.
- `/settings/import`: future import settings placeholder.
