# V1 Scope Boundaries

## Exact V1 boundaries
- Mobile app shell with Expo Router route map.
- Placeholder screens for auth, tabs, and required detail/settings routes.
- Project folder structure for app, docs, and future Supabase assets.
- No backend logic and no external API integrations.

## What not to build yet
- Supabase auth/data implementation.
- Social/community features.
- Recommendation engine.
- Subscription/billing setup.
- Analytics instrumentation.
- Non-essential abstractions or architecture expansions.
