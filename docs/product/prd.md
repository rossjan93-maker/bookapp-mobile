# Product Requirements Document (PRD)

## Product purpose
Build a solo-first, mobile reading tracker that is faster and simpler for daily use than Goodreads or StoryGraph.

## Target user
A single reader who wants a clean personal workflow for tracking books, notes, and reading activity without social complexity.

## 5 core moments
1. Open app and land in a clear home shell.
2. Find books from the search screen.
3. Keep a personal library of saved/active/finished books.
4. Write and revisit personal notes.
5. Manage account/profile basics.

## V1 in scope
- Onboarding/auth shell (placeholder only)
- Home placeholder
- Search placeholder
- Library placeholder
- Notes placeholder
- Profile placeholder
- Route and documentation foundation

## V1 out of scope
- Social feed
- Comments
- Friend graph
- Book clubs
- AI recommendations
- Subscriptions
- Analytics integrations
- Revenue tooling
