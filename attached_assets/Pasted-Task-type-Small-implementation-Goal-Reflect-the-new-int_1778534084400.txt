2. Add safe TasteReadout chip/copy support for q_outcome.
   Example chip/copy directions:

   - craft_first → “Reading for: depth”
   - originality_first → “Reading for: surprise”

3. Add safe TasteReadout chip/copy support for q_tone.
   Example chip/copy directions:
   - dark_tone → “Tone: darker”
   - light_tone → “Tone: lighter”

7. Avoid duplicate or contradictory chips where possible.
