Fix genre selection persistence across the Fiction / Nonfiction / Both segmented control during onboarding intake.

The segmented control should change which genre chips are visible, not erase selected genres.

- Continuing should save the full selected set into reader_preferences.favorite_genres.
- Avoid-genres step should exclude the full likedGenres selection across all tabs.
