- but if the page is refreshed, it does not reliably remain as the final onboarding step

Diagnosis:
The import screen is likely being shown from transient local UI state (`showImportOb = true`) rather than a persisted “pending final onboarding step” state.
That means refresh wipes the in-memory flag and the app no longer knows the user is still mid-final-step.
