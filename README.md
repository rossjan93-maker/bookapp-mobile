# bookapp-mobile

Mobile-first reading app MVP scaffold using Expo + React Native + TypeScript + Expo Router.

## Requirements
- Node.js 20+
- npm 10+

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Expo dev server:
   ```bash
   npm run start
   ```
3. Open on device:
   - Scan QR code in Expo Go, or
   - Press `a` for Android emulator, `i` for iOS simulator, `w` for web.

## Available scripts
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
